# Task 2 Perimeter
## Objective
Demonstrate your ability to optimize code and execute repetitive geometric maneuvers using a `for` loop.

![Beginner](https://img.shields.io/badge/Difficulty-Beginner-green)

## Assignment
Mission Control requires the rover to map a perfect square zone on the lunar surface. The side of the square perimeter is exactly **30 cm**. 

While you could write the movement and turning commands four separate times, a professional engineer writes efficient code. You must automate this process.

**Requirements:**
1. **The Loop:** Use a `for` loop to repeat the necessary actions exactly **4 times**.
2. **Efficiency Check:** Your program must not contain any duplicate movement commands outside of the loop.

## Conclusion
Mission accomplished! By utilizing loops, you save memory space and make your code significantly easier to read and modify.
